1. Compose

Describe your models in a simple way and choose a them from Bootswatch. 

Project Name (Theme)
--------------------
Foo
  foo1*
  foo2
  ...
Bar
  bar1*
  bar2: type
  foo_key: [foo-1, foo-2, ...]
  ...
...
  • Project name could be anything
  • Theme should be Bootswatch
  • Models should use CamelCase
  • Properties should use snake_case
  • If the field is required use *
  • If the type is not obvious : type
